# genWorkshop — 創造你的宇宙圖形｜生成藝術親子工作坊

一套給小學中年級小朋友的「生成藝術 × 尺規作圖」互動投影片（單一 HTML 檔，鍵盤左右鍵或下方按鈕切換）。

## 🔗 線上播放

**https://ileivoivm.github.io/genWorkshop/**

（透過 GitHub Pages 發布，直接用瀏覽器開啟即可播放；用 `#頁碼` 可跳頁，例如 `…/genWorkshop/#8`。）

## 內容

- 生成藝術的介紹：用「規則」創作（重複、對稱、條件變化），Sol LeWitt 的創作精神。
- 認識生成藝術之母 **Vera Molnár**，並用 p5.js 小作品示範「花朵」「迴圈 Loop」「遞迴 Recursion」。
- 伊斯蘭幾何與六角星、五芒星的故事與**尺規作圖**步驟。
- 互動「小小作圖板」：直線／同線／圓圈／同圓／多邊形，支援上一步、按住 A 鎖水平垂直、兩點求中線。
- 指令卡產生器小遊戲，最後到戶外地板用粉筆、棉線、紙膠帶創作。

## 使用方式

用瀏覽器開啟 `index.html` 即可。建議用本機伺服器（例如 VS Code Live Server）開啟，以正常載入 YouTube 影片與 p5.js。

- 網址 `#頁碼` 可直接跳頁，例如 `index.html#8`。
- 「課程流程」那頁是可點選的目錄。

## 檔案

- `index.html` — 投影片主檔
- `cover.png`、`s1`–`s6`、`倒五芒星.jpg` — 圖片素材
- `六角星的故事_文章.md` — 補充文章
